# 実装ロードマップ

原体験から逆算した機能実装の優先順位。

---

## MVP（v0.1〜v0.2：原体験の最小形）

### 完了済み
- [x] 試合（予定/結果）一覧が見られる
- [x] 試合詳細に「自分の観戦記録（観戦済み/予定）」を紐づけられる
- [x] 観戦記録にメモと費用合計を保存できる
- [x] 集計で「観戦数」「合計費用」「1試合あたり」を見られる
- [x] 費用の内訳カテゴリ（交通/チケット/飲食/その他）
- [x] Freeプラン制限（7件まで）
- [x] Plus/Proプラン対応（Stripe決済）

---

## 次期（v0.2+：原体験を"最高"に近づける）

### 高優先度
- [ ] CSVエクスポート（Proプラン）
- [ ] 複数シーズン管理（Proプラン）
- [ ] 高度集計：カテゴリ内訳グラフ
- [ ] 高度集計：月別推移グラフ

### 中優先度
- [ ] 画像添付（思い出の写真）※上限と圧縮あり
- [ ] 移動手段（電車/バス/車/飛行機/徒歩など）と所要メモ
- [ ] 宿情報（宿泊あり/なし、地域、メモ）
- [ ] 検索/フィルタ（年度・ホーム/アウェイ・相手・費用レンジ）

### 長期
- [ ] モバイルアプリ化（PWA強化またはネイティブ）
- [ ] オフライン対応
- [ ] SNS連携（Xへの投稿など）
- [ ] 複数ユーザー/共有機能
