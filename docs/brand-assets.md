# ブランドアセット ガイドライン

## 概要

本ドキュメントはOshikaで使用する画像・ロゴ・クラブ名に関するガイドラインです。
権利事故を防ぐため、以下のルールを遵守してください。

## 1. ロゴ・エンブレムに関するルール

### 禁止事項
- **フェイクエンブレムの使用禁止**: AIで生成したクラブロゴ風の画像は使用しない
- **実クラブのエンブレムの無断使用禁止**: 公式の許可なくエンブレムを使用しない
- **改変エンブレムの禁止**: 実在するクラブのエンブレムを改変した画像は使用しない

### 許可される使用
- 独自デザインのOshikaロゴ
- クラブ名のテキスト表記（「横浜F・マリノス」等）
- 公式に許可された素材（該当する場合）

## 2. LP・アプリ画像のルール

### 基本方針
- **画像内テキストは最小限**: 重要なコピーはHTML/CSSで表現
- **スクリーンショット・モックは「文字なし」**: 画面デザインを示す際は実データなしのモックを使用

### 理由
- 文字化け・多言語対応の問題を回避
- SEO/アクセシビリティの確保
- 更新時の手間を削減

## 3. スクリーンショット作成ルール

### 推奨
- 実際のUIをベースにした簡略化されたモック
- 個人情報や実データを含まないサンプルデータ
- シンプルなiOS/Androidフレーム

### 禁止
- 実在する選手名・個人名の使用
- 実際のユーザーデータの使用
- 他サービスのUI要素のコピー

## 4. クラブ名・選手名の使用

### 許可される使用
- クラブの正式名称をテキストで記載（「横浜F・マリノス」）
- 試合情報の表示（公開情報に基づく）

### 注意事項
- 公式スポンサーとの誤認を招く表現は避ける
- 「公式」「オフィシャル」等の誤解を招く表現は使用しない

## 5. 使用可能な画像一覧

### 自社作成アセット
- `/client/public/icons/` - アプリアイコン
- `/client/public/lp/` - LP用画像
- `/attached_assets/generated_images/` - 生成画像（ガイドライン準拠のもの）

### 外部素材
- ライセンス確認済みのストック画像のみ使用可

## 6. チェックリスト

新しい画像を追加する際は以下を確認：

- [ ] フェイクエンブレム・ロゴを含んでいないか
- [ ] 実在するクラブのエンブレムを無断使用していないか
- [ ] 画像内に重要なテキストが含まれていないか
- [ ] 個人情報・実データが含まれていないか
- [ ] ライセンス・権利関係は確認済みか
